These are fairly straightforward. `parseUpdate` takes in the payload from the request, and strips out any keys that aren't name, quantity, or price. Then it just simply returns the trimmed object if there's still keys left, and null if not. `parseId` is even simpler and looks for an `id` property on the payload, converts it to a number if necessary, and returns.
