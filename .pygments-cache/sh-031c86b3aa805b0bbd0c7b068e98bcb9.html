## Database Setup

For this tutorial, we will be using [SQLite3](https://www.sqlite.org/). Why? It's part of the [Python standard library](https://docs.python.org/3.5/library/sqlite3.html), requires little (if any) configuration, and is powerful enough for small to mid-size applications (e.g., the majority of web apps).
